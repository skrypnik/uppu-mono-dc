## Утилита для управления устройством "УППУ МОНО DC"

### Сборка исходного кода (Linux)

*# Создать папку для сборки*

**mkdir [путь к папке исходного кода]/uppu-mono-dc/.build**

*# Перейти папку для сборки*

**cd [путь к папке исходного кода]/uppu-mono-dc/.build**

*# Выполнить cmake, указав путь к библиотекам Qt*

**cmake .. -DQt5_DIR = [путь к корневой папке Qt]/gcc_64/lib/cmake/Qt5**

*# Собрать утилиту*

**make**
